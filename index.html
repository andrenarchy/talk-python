Pakete: numpy

numpy ist das Basispaket für Numerik (baut auf BLAS, LAPACK, usw. auf).

from numpy import array, ones, dot   # N-dim Arrays
v = array([1,2,3])           # 1-dim arr, shape==(3,)
v = ones(5)                  # 1-dim arr, shape==(5,)
v = ones((5,1))              # 2-dim arr, shape==(5,1)
A = array([[1,2,3],[4,5,6]]) # 2x3 Matrix: 2-dim arr, shape==(2,3)
A = ones((5,5))              # 5x5 Matrix: 2-dim arr, shape==(5,5)
b = dot(A, v)                # A*v

Der Standard-Datentyp in numpy sind numpy arrays. Es gibt auch noch numpy matrix: der wesentliche Unterschied ist, dass man wie in MATLAB den *-Operator für die Matrix-Matrix-Multiplikation benutzen kann. Wir (und die numpy community) empfehlen numpy arrays.

from dolfin import *
mesh = UnitSquareMesh(32, 32)
V = FunctionSpace(mesh, "Lagrange", 1)
u, v = TrialFunction(V), TestFunction(V)
f = Expression("exp(-(pow(x[0] - 0.5, 2) + pow(x[1] - 0.5, 2)))")
a = inner(grad(u), grad(v))*dx
L = f*v*dx
bc = DirichletBC(V, Constant(0.0), lambda x, on_bd: on_bd)
u = Function(V)
solve(a == L, u, bc)
plot(u, interactive=True)
